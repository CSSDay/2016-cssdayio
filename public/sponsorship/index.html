About CSSDay.io

CSSDay.io is an all-day event on Saturday, December 3th 2016. The audience will be composed of many skill levels. However, we are focusing on topics that will appeal to seasoned developers, so you can expect the event to be a great networking opportunity for your company. All web developers eventually work with CSS even if their core competencies are programming. This means if you're looking to network with JavaScript, Ruby, Python, Java, or .NET developers, you'll find them all here at CSSDay.io

Last year's event surpassed expectations with approximately 300 guests in attendance. Check out some of the awesome photos we were able to capture

Event Highlights

The event features:

  • All-day talks for CSS professionals
  • Free Lunch (see sponsorship opportunity)
  • Swag and T-Shirt Giveaways
  • An After Hours Party! (see sponsorship opportunity)
